# Semilla Website UI Kit

A hi-fi, click-through website prototype for Semilla Café & Coworking.

## Design width
1440px desktop, responsive down to mobile.

## Screens / Sections
1. **Header** — sticky nav with logo + links + CTA
2. **Hero** — full-height hero with logo mark, headline, CTA
3. **Coffice section** — dual café + coworking proposition
4. **Coworking Plans** — Flex / Full / Office pricing cards
5. **Menu preview** — café offerings teaser
6. **Community/Events** — upcoming events strip
7. **Footer** — contact, social, address

## Files
- `index.html` — main entry, loads all components
- `Header.jsx` — sticky navigation
- `Hero.jsx` — hero section
- `Sections.jsx` — Coffice, Plans, Menu, Events
- `Footer.jsx` — footer
